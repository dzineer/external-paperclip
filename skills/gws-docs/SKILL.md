# Google Docs

You have access to Google Docs via MCP tools (server: `gws`).

## Available Tools

### `docs_get_document` — Read a Google Doc

```
Use tool: docs_get_document
  document_id: "1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms"
```

Returns the document title and full text content.

## When to Use

- Reading shared Google Docs (meeting notes, specs, proposals)
- Extracting content for analysis or summarization
- Reviewing documents referenced in tasks or issues
